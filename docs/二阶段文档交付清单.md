## 4. 文档覆盖的功能范围

本次文档覆盖以下二阶段功能：

- 足球形态校园智能助手。
- 主 Agent + 子 Agent 能力路由。
- 自然语言教务查询和权限内增删改查。
- 请假考勤模块。
- 综合知识库 RAG。
- 学业辅导和诗词鉴赏。
- 情绪陪伴和人在回路提醒。
- 文档处理、OCR、英汉互译。
- 搜索引擎、路线生活、世界杯问答、AI 知识问答。
- 编程助手和 GitHub 助手。
- 权限控制、二次确认、短期上下文、任务草稿和长期记忆。
